# 🛰️ Nivel 17: Omnipresencia Predictiva (Intent Matrix)

El Nivel 17 representa la **"Cúspide de Inteligencia"** de Richard Automotive. Ya no solo nos adaptamos al perfil psicológico (Nivel 16), sino que **predecimos la intención de compra** con una precisión milimétrica utilizando análisis semántico de última generación.

---

## 🏗️ 1. Matriz de Intención Semántica (LIMP)

Utilizamos el `IntentAnalysisService` alimentado por **Gemini 1.5 Flash** para extraer una matriz de 4 dimensiones por cada interacción:

- **Urgencia (0.0 - 1.0)**: ¿El cliente necesita el vehículo de inmediato?
- **Sensibilidad al Precio (0.0 - 1.0)**: ¿El factor decisivo es el ahorro o el valor?
- **Lealtad a Marca (0.0 - 1.0)**: ¿Qué tan flexible es el cliente respecto al inventario?
- **Madurez de Decisión (0.0 - 1.0)**: ¿En qué etapa del embudo de ventas se encuentra?

## ⚡ 2. Sentiment Velocity (Calor de Venta)

El sistema calcula la **Velocidad de Sentimiento** comparando las dos últimas matrices del lead. Esto permite detectar si el interés está aumentando o si el lead se está enfriando, disparando alertas proactivas al equipo de ventas.

## 🎯 3. Arbitraje de Intención en UI

El `PersuasionWrapper` evoluciona para manejar **Casos de Urgencia Crítica**. Si la matriz detecta una urgencia > 0.8, el sistema ignora las variantes genéricas y despliega un **"Acceso de Alta Prioridad"** (ej. Botón de WhatsApp Directo con Richard).

---

## 🏁 Compromiso de Omnipresencia

En Richard Automotive, la omnipresencia no es invasiva; es **Oportuna**. El sistema sabe qué mostrar, cuándo mostrarlo y por qué, eliminando el ruido y enfocándose exclusivamente en lo que el cliente realmente necesita para tomar una decisión informada.
