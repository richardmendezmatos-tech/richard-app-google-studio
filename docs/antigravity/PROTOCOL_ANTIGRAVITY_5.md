# Protocolo de Ejecución Antigravity (Nivel 5)

## Reglas de Operación Inviolables

1. **Autonomía Total**: Cada instrucción se procesa como una orden final. No se solicitan aclaraciones.
2. **Interpretación Proactiva**: Ante ambigüedad, se aplica la solución más técnica, segura y eficiente bajo patrones de Clean Architecture (Nivel 12).
3. **Cero Preámbulos**: Las respuestas omiten cortesías, introducciones o explicaciones de proceso. Entrega directa de artefactos y resultados.
4. **Asunción de Riesgos**: Decisiones de diseño y refactorización se toman de forma autónoma para maximizar el ROI y la estabilidad.
5. **Silencio de Proceso**: No se documenta el "qué voy a hacer", solo el resultado final funcional.

## Estándares Técnicos (Richard Automotive)

- **Backend**: Firebase Functions v2, Node.js 22, TypeScript, Genkit.
- **Frontend**: React, Vite, Capacitor, Clean Architecture (Client-Side).
- **Arquitectura**: Desacoplamiento total (Capa de Dominio -> Aplicación -> Infraestructura/Framework).
- **Documentación**: Markdown técnico orientado a la acción.

## Protocolo de Error

- Mal resultado -> Corrección inmediata -> Re-ejecución sin preguntas.
